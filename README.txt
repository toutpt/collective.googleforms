Introduction
============

Google provides a service which let you create forms_. This add-on let you use
a form within Plone throw a view over a Link content type.

How it works
============

1.This add-on parse the html of the form you have built in google doc.
2.It build a form with the same html markup has archetypes, so the form seems
to be a plone one.
3.The action is the google doc one, so you are redirected to it.
